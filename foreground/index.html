Grafana Wapp

Introduction

The porpuse of this wapp is to send the values of a Wappsto device to a Grafana server. To do this you need enter the full name of the device below, and click 'Set name'. Then a request will be sent to get that device. The first time this is done you need to accept the permissions of that device.

A permissions given to this application can be reset by uninstalling/reinstalling it in Wappsto.

The device can be changed in order to get different data. Simply enter a new name below.
